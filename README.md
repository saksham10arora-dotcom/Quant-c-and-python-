# Quantitative Finance & Systems Engineering Mastery

This repository is a unified, comprehensive toolkit documenting my research and implementations across two distinct but deeply connected pillars of modern quantitative finance: **High-Performance C++ Systems** and **Python-based Quantitative Research & Machine Learning**.

## 🏗️ Repository Architecture

The project is split into two primary domains, each focusing on specific aspects of the quantitative trading stack.

### 1. `cpp_mastery/` - Low-Latency & Core Infrastructure
Focused on building the robust, highly-optimized infrastructure required for high-frequency trading (HFT) and complex systems.

- **`01_Core_Language/`**: Deep dives into modern C++ memory management (Smart Pointers) and Move Semantics.
- **`03_Concurrency/`**: Thread management, Mutexes, Atomics, and Memory Ordering for safe parallel execution.
- **`04_Low_Latency_HFT/`**: Techniques for squeezing out microseconds, including SIMD Vectorization (AVX) and Lock-Free Data Structures (SPSC Ring Buffers).
- **`05_Trading_System_Components/`**: Core financial structures, featuring a highly optimized Limit Order Book (LOB) implementation.

### 2. `python_quant/` - Quantitative Research & Deep Learning
Focused on alpha generation, data analysis, and portfolio optimization.

- **`02_Data_Science_Stack/`**: Efficient vectorized data manipulation using NumPy and Pandas for large-scale financial datasets.
- **`03_Machine_Learning/`**: Deep learning models for time-series forecasting, featuring PyTorch-based LSTM architectures.
- **`04_Quantitative_Finance/`**: Implementations of Modern Portfolio Theory, Efficient Frontier mapping, and Tangency Portfolio optimization.
- **`05_Algorithmic_Trading/`**: A custom, event-driven backtesting engine capable of simulating strategies (like SMA Crossovers) with realistic market mechanics.

## 🚀 Tech Stack
- **C++**: C++17/20, STL, GCC intrinsics (SIMD), Lock-free programming paradigms.
- **Python**: Python 3.10+, NumPy, Pandas, SciPy.
- **Deep Learning**: PyTorch.

## 🎯 Objective
This repository serves as both a theoretical reference and a practical codebase. It bridges the gap between mathematically rigorous trading strategies designed in Python and the ultra-low-latency execution engines required to deploy them in C++.
